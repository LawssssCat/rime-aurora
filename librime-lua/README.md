备份 rime.dll 

>使用说明：
>
> 确定使用哪个版本，把其名字改成 `rime.dll`，然后覆盖到程序目录即可。

>版本说明：
>
> - `2655a30` (hchunhui) 支持 log 方法
> - `754aaf4` (LawssssCat) 支持 regex_match 方法
> - `1a35083` (LawssssCat) 支持 opencc convert_word 方法
